### Importing Applications

<Row>
    <Col>

        In the import app modal, you can import an app from a file with the extension `.zip`.
        During the process of importing an application, if the message-types folder is present in the application, the
        message types will be imported to the message center with the specified templates same as all the process models
        and decision tables.

    </Col>
    <Col>
        <ImagePreview
            src="/images/apps/actions/import-app-modal.png" alt="Import app modal"/>
    </Col>
</Row>

<Row>
    <Col>

    </Col>

        exists in the system, a conflict will be detected, and the user will be prompted to pick a strategy to resolve
        the conflict.

        After choosing a strategy, a table will appears with a summary of how conflicts were resolved.
