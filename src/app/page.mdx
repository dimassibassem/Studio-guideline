# Bankerise Studio Documentation

The new Bankerise Studio is a web-based application that allows users to create, edit, and manage forms and BPMN models.

It is built on top of the Flowable Modeler and Formio,
and it is designed to be a user-friendly and intuitive tool for creating and managing forms and BPMN models.

Significant changes have been implemented to enhance functionality, security, and user experience.
