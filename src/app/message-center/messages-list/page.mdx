# Message Templates
<Row>
    <Col>
        The Message Templates page displays all the message templates that have been created.
        A table displays the message templates grouped by category. You can search for a specific message template using
        the search bar.

        The displayed data is paginated, and you can navigate through the pages using the pagination controls.
        Clicking on a message template will take you to the Message Template editor page, where you can see the message
        and edit it.

<Row>
    <Col>

    </Col>
    <Col>

        In the table there is `Actions` column where you can:
        - See the message template
        - Duplicate the message template
        - Download the message template
        - Delete the message template
    </Col>
</Row>

You will find also in the table all available channels with a toggle to enable/disable the channel for the message template.

The `Status` column shows the status of the message template, whether it is:

- `Completed` when all channels are configured.
- `Incomplete` when at least one channel is not configured.
- `Empty` when no channel is configured.
