# Template Editor

The Template editor is where you can create and edit message templates in Bankerise Studio.

<Row>
    <Col>

        The template editor allows you to create and edit message templates for various channels in multiple languages.

        You can use `Freemarker` code to add dynamic content and subject to your message templates.

        As you type, the editor provides auto-completion for Freemarker syntax and schema variables. On the right side
        of the editor, a live preview displays how your message template will appear.

        On the left side, you'll find an object containing schema variables along with fake data, which you can use to
        visualize how the final message will look.

    </Col>
