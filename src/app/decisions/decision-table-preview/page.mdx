# Decision table preview
After clicking on a decision table in the decision tables list, you will be taken to the decision table preview page. Here you can see the decision table in more detail.

<Row>
    <Col>
        <ImagePreview
            src="/images/decisions/decision-preview.png" alt="Decision table preview"/>
    </Col>
    <Col>

        The Decision table preview page displays the decision table in more detail. You can see the decision table name,
        version, key and the
        date it was last modified.

        You can also see the decision table preview image, which shows the decision table fields .

        You can use the buttons at the top of the page to edit the decision table, view the decision table history, or
        delete the
        decision table.
    </Col>
</Row>
